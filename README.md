# NewGold - H5网页游戏化金融产品

## 🎯 项目简介

这是一个基于H5网页的7天周期性任务游戏平台，集成了红包抢夺、任务系统和团队裂变机制。

### 核心功能
- 🎮 **新手任务系统** - 引导用户完成基础操作
- 🧧 **抢红包功能** - 每日三场定时红包抢夺
- 👥 **团队邀请机制** - 多层级用户结构
- 💰 **钱包系统** - 资金管理和提现
- 🏆 **排行榜** - 多维度用户排名
- 📱 **响应式设计** - 完美适配移动端

### 技术栈
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **后端**: Node.js + Express
- **数据库**: 支持多种数据库
- **部署**: Python HTTP服务器

## 🚀 快速开始

### 环境要求
- Node.js >= 16.0.0
- Python 3.x

### 安装运行

1. **克隆项目**
```bash
git clone https://github.com/mason63771177/newgold.git
cd newgold
```

2. **启动后端服务**
```bash
cd backend
npm install
node server.js
```

3. **启动前端服务**
```bash
# 在项目根目录
python3 -m http.server 3001
```

4. **访问应用**
打开浏览器访问: http://localhost:3001

## 📱 功能特性

### 用户状态系统
- **状态1**: 新用户/未激活 - 显示激活账号、排行榜
- **状态2**: 已激活/168小时倒计时 - 显示邀请链接、任务、团队、抢红包、钱包、排行榜
- **状态3**: 倒计时结束/未复购 - 显示再次激活、团队、钱包、排行榜
- **状态4**: 挑战完成状态 - 显示邀请链接、大神任务、团队、抢红包、钱包、排行榜

### 任务系统
- **新手任务**: 1-3 按序完成
- **答题任务**: 完成新手任务1后开放
- **大神任务**: 完成新手+答题任务后开放

### 抢红包系统
- **时间窗口**: 每日 9:00 / 12:00 / 20:00
- **抢夺时长**: 每场77秒
- **动态分配**: 智能金额分配算法
